# MyDemands - Redigir com IA

## Instalação
```bash
pip install -r requirements.txt
```

## Configurar chave da OpenAI
```bash
export OPENAI_API_KEY="sua-chave"
```

## Rodar testes
```bash
pytest
```

## Habilitar/desabilitar IA no app
- Abra o app e clique no botão **✨** (à esquerda de Notificações).
- Em **Configurações da IA**, você pode habilitar/desabilitar o recurso, modelo e temperatura.
- Sem `OPENAI_API_KEY`, o botão de **Redigir com IA** fica desabilitado.
